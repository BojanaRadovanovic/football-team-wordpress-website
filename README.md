# football-team-wordpress-website

This project is a WordPress-based website dedicated to the Serbia national football team.

## Features
- Responsive design
- Dynamic content management using WordPress
- Sections for players, matches, and team information
- Easy content updates through admin panel

## Technologies
- WordPress
- PHP
- HTML
- CSS

## Description
This project represents a dynamic version of a football team website built using WordPress. It allows easier content management compared to static websites, enabling updates through the admin dashboard without modifying the code directly.

## Setup Instructions
- Install WordPress on a local server (XAMPP/WAMP)
- Place the project files in the server directory (htdocs)
- Run the project via browser (localhost)

## Learning Outcome
This project helped me understand content management systems (CMS), theme customization, and working with dynamic web content.
